This was written by Aaron Rosenbaum as a proof of concept for sliding semantically correct navigation.

This project is hosted on  Github.

For this snippet, I placed the <nav> tag inside the header and position it to the left of #content with CSS and animate the transition. It only activates the alternate layout when viewed from a phone (480px wide or less).

CSS3 was used for the animation and shadows. This solution should work in all modern browsers, along with IE8 and higher.
